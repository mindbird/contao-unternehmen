contao-unternehmen
==================

This bundle provides a Contao CMS integration to manage and display company information.

## Features
- Manage company details such as name, address, contact information, and logo.
- Display company information using Contao content elements.
- Easy integration with Contao CMS.

## Configuration
1. Install the bundle via Composer:
   ```bash
   composer require mindbird/contao-unternehmen
   ```
2. Clear the Contao cache and run the database update tool if necessary.

## Usage
- Use the provided Contao content elements to display company information on your website.
- Configure company details in the Contao backend under the "Company" section.
